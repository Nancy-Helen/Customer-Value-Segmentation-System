# Customer Value Segmentation System

A Machine Learning-based customer segmentation application that identifies high-value customers using RFM (Recency, Frequency, Monetary) analysis and predicts Customer Lifetime Value (CLV). The application provides interactive visualizations and business insights through a Streamlit dashboard.

---

## Project Overview

This project helps businesses analyze customer purchasing behavior and segment customers based on their value. By applying RFM analysis and machine learning techniques, the system enables organizations to identify their most valuable customers, improve retention strategies, and make data-driven marketing decisions.

---

## Features

- Upload customer transaction datasets
- Data preprocessing and cleaning
- RFM (Recency, Frequency, Monetary) analysis
- Customer segmentation
- Customer Lifetime Value (CLV) prediction
- Interactive Streamlit dashboard
- Business insights and visualizations
- Download processed results

## Workflow

1. Upload customer transaction dataset.
2. Clean and preprocess the data.
3. Calculate Recency, Frequency, and Monetary (RFM) values.
4. Segment customers based on RFM scores.
5. Predict Customer Lifetime Value using a machine learning model.
6. Display customer insights and visualizations through the Streamlit dashboard.

## Installation

Clone the repository:

```bash
git clone https://github.com/mansim20/Customer-Value-Segmentation.git
```

Navigate to the project folder:

```bash
cd Customer-Value-Segmentation
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
